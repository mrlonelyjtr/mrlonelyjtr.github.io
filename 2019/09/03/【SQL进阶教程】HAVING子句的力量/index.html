用HAVING子句进行自连接：求中位数

前面的表Graduates里有10行数据，所以取“史密斯，20000”和“劳伦斯，15000”的平均值17500作为中位数。来思考一下如何来查询位于集合正中间的元素。
做法是，将集合里的元素按照大小分为上半部分和下半部分两个子集，同时让这2个子集共同拥有集合正中间的元素。这样，共同部分的元素的平均值就是中位数。
